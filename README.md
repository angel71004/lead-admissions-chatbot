# LEAD College Admissions Chatbot

A web-based chatbot for MCA and MBA admissions that collects student details through a conversational interface and exports application data as CSV files.

## Features
- Interactive chatbot flow
- MCA / MBA application support
- Input validation
- Automatic CSV download
- LocalStorage support
- Responsive UI

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)
- Python 3.11

## Usage
1. Open index.html in a browser
2. Answer chatbot questions
3. Download application CSV automatically
